## CUDA setup 
1. Nvidia Graphics driver installation
  $ sudo add-apt-repository ppa:graphics-drivers/ppa
  $ sudo apt-get update
  $ sudo apt-get install nvidia-375

  After this, reboot 
  $ sudo reboot
  
  When u type  $ nvidia-smi   , you should see the following message.
